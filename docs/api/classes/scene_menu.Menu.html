constructor

  • new Menu(context: Context, elementId: string, title: string, isGlobal?: boolean): Menu

  • All menus should inherit from this class. Registers the menu with main context and allows it to be activated, inactivated, and to handle hotkeys. Also gives it access to the 3d scene.

    Parameters

    • context: Context

      main context

    • elementId: string

      the id of the html element containing this menu

    • title: string

      the name used when referring to this menu

    • isGlobal: boolean = true

      True if never loses focus. If false, inactivate() is called when losing focus and activate() when gaining it.

    Returns Menu
